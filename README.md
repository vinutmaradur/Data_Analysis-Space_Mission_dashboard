# Global Space Missions Dashboard

# Project Overview 📌

The Global Space Missions Dashboard is an interactive Tableau project that analyzes historical space missions across different countries, companies, and rockets. It provides deep insights into mission success rates, launch trends, and key players in the global space industry.

This dashboard helps users explore patterns in space exploration and understand performance metrics of rockets and organizations.

---

# Objectives 🎯
- Analyze global space mission trends over time
- Compare success vs failure rates
- Identify top-performing rockets and organizations
- Visualize launch locations across the world
- Provide data-driven insights for space industry analysis

---

# Dashboard Features
KPI Metrics
- Total Missions: 4,630
- Successful Missions: 4,162
- Failed Missions: 357
- Success Rate: 90%
- Total Companies: 62
- Launch Locations: 158

Key Visualizations
📈 Space Missions Over Time
- Shows trends in mission launches from 1950s to present
- Identifies growth phases and peak activity periods
🚀 Rocket Usage Analysis
- Displays most frequently used rockets
- Includes success rate comparison
✅ Rocket Success Rate
- Highlights high-performing rockets (e.g., Falcon 9 Block 5 with 100%)
🌍 Mission Launch Map
- Geographic distribution of launch sites worldwide
🏢 Top Launch Organizations
- Countries/companies with highest number of missions
- USSR/Russia and USA dominate
📊 Success vs Failure Distribution
- Majority missions are successful (~90%)
- Small proportion of failures and partial failures
📍 Launch Sites Analysis
- Most active launch sites like Baikonur Cosmodrome
📅 Launch Timeline
- Tracks missions over years by type
